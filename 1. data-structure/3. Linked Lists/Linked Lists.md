# Linked List

A linked list is `a linear collection of data elements` called nodes `each pointing to the next node` by means of a pointer. Javascript doesn't have a built-in Linked List

## # Pros of Linked List

- Fast Insertion
- Fast Deletion
- Ordered
- Flexible Size

## # Cons of Linked List

- Slow Lookup
- More Memory
